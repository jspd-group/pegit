Help Manual -CREATE COMMAND

Name

peg-create - Creates an empty repository or reinitialize an existing one.

Syntax

peg create [project-name]

Description

Peg create command - This command creates an empty Pegit repository - basically a .peg directory with subdirectories for objects, refs/heads, refs/tags, and template files.
Peg- create is the first command , that a user is supposed to give before connecting his/her project with the pegit database. It initializes the contents of the database with default values and adds a new entry corresponding to the given project name in the database of the project.
Running the peg-create command on already existing pegit repository is safe and will not duplicate the pegit repositories.

Examples

  • Start a new pegit repostory for the given source code base.
    $ cd
 path/to/your/source-code
 
$ peg create project_name
# you can provide any name for your project
