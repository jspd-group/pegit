Help Manual - ALL COMMANDS


List

  • PEG HELP ABOUT : Tells what is PEGIT all about
  • PEG CREATE "project_name" : Create a PEG repository
  • PEG INSERT : Insert the files intp stage area or temporary database
  • PEG COMMIT : Commit the changes.
  • PEG RESET : Remove the files present in the stage area
  • PEG HIST : Logs the commit.
  • PEG HEAD : Change the position of head pointer.
  • PEG STATUS : Shows the status of the project directory.
  • PEG SHOW : Displays the table.
  • PEG REVERT : Revert the local changes.
  • PEG COMPARE : Shows the changes made till last commit.
  • PEG CHECKOUT : Apply the version change command.
  • PEG LIST : Displays the file in last commit.
  • PEG TAG : Apply a tag to the commit.
  • PEG HELP : Shows the help for the command.

Note

  • Type PEG HELP "command" to get detailed information on a particular command
